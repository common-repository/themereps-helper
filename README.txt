== Description ==

Themereps Helper is a companion plugin for Themereps Themes, which provides core functionality and extends free and premium themes features. The plugin requires the [Advanced Import : One Click Import for WordPress or Theme Demo Data](https://wordpress.org/plugins/advanced-import/) plugin to be installed for demo data importing in just a click.

How to use this Themereps Helper?

1. While installing free or premium themes by themereps do kindly install & activate the plugin "Themereps Helper"
2. The plugin requires the [Advanced Import : One Click Import for WordPress or Theme Demo Data](https://wordpress.org/plugins/advanced-import/) plugin to be installed to demo data import. So install and active it.
2. To import demo contents, please go to Appearance > Demo Import.
3. Choose your demo by checking live preview and then click import button to import the demo.
5. Feel free to contact themereps support team at ( https://themereps.com/contact/ ) if you need any assistance.


== Installation ==

You can install Themereps Helper Plugins as following two ways:
1. Upload zip file from Dashboard->Plugin->Add New "upload plugin" or
2. Extract Themereps Helper and placed it to the "/wp-content/plugins/" directory.
    - Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= Is Themereps Helper is a free plugin? =

Yes, it is a free plugin.

= Does this plugin works with any WordPress theme =

No, This plugin will only work with the themes developed by [Themereps](https://themereps.com/).
